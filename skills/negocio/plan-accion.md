# Skill: Plan de Acción

## Activación
Palabras clave: `plan`, `estrategia`, `evaluar idea`, `cómo implemento`

## Estructura obligatoria
1. **Objetivo concreto** (1 línea, medible)
2. **Pasos ordenados** (numerados, accionables)
3. **Riesgos por paso** (1 línea cada uno)
4. **Métricas de éxito** (medibles: N usuarios, T tiempo, % tasa)
5. **Tiempo estimado** (por paso y total)

## Formato
```
🎯 Objetivo: [frase]

📋 Pasos:
1. [acción] — riesgo: [x] — tiempo: [Xh]
2. ...

📊 Métricas éxito:
- 

⏱️ Total estimado: [Xh / Xd]
```

## NO
- Generar >10 pasos sin aprobación Anthony
- Proponer herramientas nuevas sin justificar vs stack actual
- Inventar métricas genéricas (`mejor UX`) sin número concreto
