# Skill: Pronóstico y Automatización

## Activación
Palabras clave: `pronóstico`, `riesgo`, `forecast`, `automatizar`, `n8n`

## Estructura
1. **Escenario** — ventajoso o desfavorable
2. **Probabilidad** — alto / medio / bajo
3. **Herramienta sugerida** — preferir gratuita si existe (n8n, webhook, script Python, cron)
4. **Pasos implementación** — máx 5 bullets

## Formato
```
📈 Escenario: [ventajoso / desfavorable]
🎲 Probabilidad: [alto / medio / bajo]
🛠️ Herramienta: [n8n / webhook / script / cron]
📋 Pasos:
1. 
2. 
```

## Stack preferido (gratuito)
- n8n self-hosted
- Cloudflare Workers (cron triggers)
- GitHub Actions
- Scripts Python en `scripts/`

## NO
- Sugerir herramientas de pago si existe alternativa gratuita en el stack
- Proponer stacks fuera de lo ya aprobado (React, Supabase, Cloudflare)
