# CTX-01 — Contexto del Proyecto

**Nombre:** CiudApp
**Descripción:** Red ciudadana B2G para Venezuela — app ciudadano (gratis) + dashboard alcaldía (pago)
**Fase:** 2 — MVP Core (semana 3-8, en curso)
**Stack:** React 18 + Vite | Supabase PostgreSQL | Cloudflare Pages | Leaflet + OpenStreetMap | Zustand | GitHub

**UX Target:**
- Ciudadano reporta incidente en 3 taps (tap mapa → mini-popup → wizard)
- 100% mobile-first — desktop muestra landing page
- Offline-first, modo oscuro, encriptado
- Dashboard alcaldía ve todos los reportes en tiempo real + responde por chat bidireccional
- Referencia UX principal: SoSafe (Chile) — aplicar selectivamente

**Roles (como Yummy pasajero/conductor):**
- Login ciudadano → app móvil
- Login alcaldía → dashboard admin (reportes + chat + estados)

**Municipio piloto:** Chacao (Caracas)
**Veredicto:** Proyecto viable. Stack correcto. Resolver 5 alertas AUDITORIA.md antes de MVP.
