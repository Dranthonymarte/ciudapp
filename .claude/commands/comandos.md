# /comandos — Listar comandos disponibles

## Proceso
`ls .claude/commands/` → listar con descripción de 1 línea cada uno

## Salida
```
📟 Comandos CiudApp

Sesión:
  /iniciar      — Abrir sesión + menú principal
  /continuar    — Retomar tarea activa
  /cerrar       — Cerrar tarea y mover a completado
  /status       — Estado del proyecto (≤20 líneas)

Trabajo:
  /nueva-tarea  — Crear tarea del roadmap
  /reunion      — Procesar transcripción de reunión
  /idea         — Agregar al BACKLOG
  /regla        — Agregar regla permanente
  /auditoria    — Auditar seguridad módulo

Ayuda:
  /comandos     — Este listado
```

## Restricciones
- NO abrir el contenido de cada comando — solo listar
