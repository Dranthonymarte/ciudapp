# /archivo

Uso: /archivo [ruta/al/archivo]

1. grep headers del archivo indicado
2. Leer solo secciones relevantes con offset+limit
3. Analizar en contexto del proyecto CiudApp
4. Reportar: resumen, relevancia para el proyecto,
   acción sugerida (mover, actualizar CTX, crear tarea, archivar)

Aplica para cualquier tipo: PDF, MD, TXT, HTML

NO leer archivo completo si >100 líneas
