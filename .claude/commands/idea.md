# /idea — Agregar al BACKLOG

## Proceso
1. Capturar idea de Anthony (1-2 líneas)
2. Append directo en `ideas/BACKLOG.md`:
   `| [YYYY-MM-DD] | [idea] | pendiente | [prioridad] |`
3. Si el archivo no existe → crearlo con header de tabla

## Header tabla (si crea el archivo)
```markdown
# BACKLOG — CiudApp

| Fecha | Idea | Estado | Prioridad |
|-------|------|--------|-----------|
```

## Restricciones
- NO abrir el archivo completo para leer — solo append
- Prioridad default: `media` si Anthony no la indica
- Confirmar en 1 línea: `✅ Idea agregada al backlog`
